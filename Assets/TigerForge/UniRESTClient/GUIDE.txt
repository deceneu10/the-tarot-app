====================
     TigerForge
  UniREST Solution
====================

- Install, in you Server solution (Web Hosting, phisical Server, etc.) WordPress.
- Install the UniREST Server plugin in WordPress (the tigerforge-unirest-server-XX.zip file in the UniRESTServer folder).
- Use the UniREST Server web application to set up your Database tables and REST APIs.

- Copy the C# Configuration Script generated by the UniREST Server app into the UniRESTClientConfig.cs file of this Asset.
- In your C# Scripts, import the TigerForge namespace and start using the UniREST Client class.

For more informations:
http://tigerforge.altervista.org/docs/unirest-1-0/
